neon-js
*******

neon-js is a Javascript library to interface with NEO blockchain.

==========

    import * as Neon from 'neon-js'

    Neon.getBalance("TestNet", "ALq7AWrhAueN6mJNqk6FHJjnsEoPRytLdW").then((response) => {
